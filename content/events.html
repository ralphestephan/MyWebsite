JavaScript Events Overview

Event Name Description Category Supported HTML Tags W3Schools Link
submit Triggers when a form is submitted Form Events <form> submit Event
change Activates when the value of an element is modified Form Events <input>, <select>, <textarea> change Event
focus Occurs when an element gains focus Form Events <input>, <select>, <textarea> focus Event
blur Triggers when an element loses focus Form Events <input>, <select>, <textarea> blur Event
input Activates when input value changes Form Events <input>, <textarea> input Event
dragstart Fires when the user starts dragging an element Drag Events <img>, <div>, etc. dragstart Event
drag Occurs while an element is being dragged Drag Events <img>, <div>, etc. drag Event
drop Triggers when an element is dropped Drag Events <img>, <div>, etc. drop Event
copy Activates when content is copied Clipboard Events <input>, <textarea> copy Event
cut Occurs when content is cut Clipboard Events <input>, <textarea> cut Event
paste Triggers when content is pasted Clipboard Events <input>, <textarea> paste Event
click Occurs when an element is clicked Mouse Events <button>, <a>, <div>, etc. click Event
dblclick Occurs when an element is double-clicked Mouse Events <button>, <a>, <div>, etc. dblclick Event
mouseover Triggers when the pointer is moved onto an element Mouse Events <div>, <a>, <button>, etc. mouseover Event
mouseout Occurs when the pointer is moved out of an element Mouse Events <div>, <a>, <button>, etc. mouseout Event
keydown Occurs when a key is pressed down Keyboard Events <input>, <textarea>, <body>, etc. keydown Event
keypress Occurs when a key is pressed and held Keyboard Events <input>, <textarea>, <body>, etc. keypress Event
keyup Triggers when a key is released Keyboard Events <input>, <textarea>, <body>, etc. keyup Event
load Fires when the page has finished loading Window Events <body>, <img>, etc. load Event
resize Fires when the browser window is resized Window Events <window> resize Event
scroll Occurs when an element is scrolled Window Events <window>, <div>, etc. scroll Event
